# zt
> Minimalistic node.js logging library

## Installation
```shell
$ npm install zt
```

## Usage
```javascript
import zt from 'zt';

zt.log('Log normally');
zt.warn('Log warning');
zt.error('Log error'); // or zt.error(new Error('Log Error');
```
